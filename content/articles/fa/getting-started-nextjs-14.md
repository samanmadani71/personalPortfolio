# شروع کار با Next.js 14

Next.js 14 ویژگی‌ها و بهبودهای جدید هیجان‌انگیزی را به فریمورک React اضافه می‌کند. در این مقاله، ویژگی‌های کلیدی و نحوه شروع کار را بررسی خواهیم کرد.

## ویژگی‌های کلیدی

### ۱. App Router
App Router جدید روشی بصری‌تر برای ساختاردهی برنامه شما با مسیریابی مبتنی بر فایل ارائه می‌دهد.

### ۲. Server Components
Server Components به شما امکان می‌دهند کامپوننت‌ها را روی سرور رندر کنید و اندازه بسته را کاهش دهید و عملکرد را بهبود بخشید.

### ۳. بهبود عملکرد
Next.js 14 شامل بهینه‌سازی‌های متعدد عملکرد به صورت پیش‌فرض است.

## شروع کار

برای ایجاد یک پروژه جدید Next.js 14:

```bash
npx create-next-app@latest my-app
cd my-app
npm run dev
```

## نتیجه‌گیری

Next.js 14 یک فریمورک قدرتمند برای ساخت برنامه‌های وب مدرن است. آن را در پروژه بعدی خود امتحان کنید!
